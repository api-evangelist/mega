# MegaThis is a repo for managing machine readable artifacts for the [Mega APIs](https://mega.co.nz). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Mega APIs](https://mega.co.nz).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://mega.co.nz)- [Developer](https://mega.co.nz)- [Twitter](https://twitter.com/MEGAprivacy)The [APIs.json](https://github.com/api-evangelist/mega/blob/master/apis.json) index provides links to all the relevant links for the Mega APIs, including the OpenAPI and Postman if they are present.
